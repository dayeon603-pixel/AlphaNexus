### State at Session Start

### Work Completed

**1. Bug fix: `tests/engine/math.test.js` — softplus "always positive" test**

`softplus(-1000)` underflows to 0 in float64 (exp(-1000) = 0 → log(1+0) = 0).
Test value changed from -1000 → -10 (exp(-10) ≈ 4.5e-5, `log(1 + exp(-10))` > 0 in float64).

**2. Bug fix: `tests/engine/risk.test.js` — constant-return uptrend fixture**

`prices = 100 * exp(0.0005 * i)` yields constant log-returns, so `std(returns) = 0`,
`annVol = max(0, 1e-12) * sqrt(252) ≈ 1.6e-11`, which `toFixed(6)` rounds to 0.
Three dependent tests then fail: `annVol > 0`, `omega=999`, `winRate=1`.

Fix: alternating log-returns (0.0003, 0.0008) via
`100 * exp(floor(i/2) * 0.0011 + (i%2===0 ? 0 : 0.0003))`.
This is strictly increasing (mdd=0, winRate=1, omega=capped) with non-zero variance.

**3. New test file: `tests/engine/fetcher.test.js` — 14 tests, 2 describe blocks**
